# Extra image refs to bundle into phantomos-k0s-images*.deb beyond
# what build-images-deb.sh's manifests/ scan picks up. One ref per
# line; blank lines and `#` comments are OK.
#
# Two ways to vary by architecture:
#
#   1. {ARCH} / {KERNEL_ARCH} template substitution
#      {ARCH}        -> debian arch:    amd64 / arm64
#      {KERNEL_ARCH} -> uname -m form:  amd64 / aarch64
#      Use this when one tag template covers every arch your CI
#      publishes for the same image.
#        foundationbot/some-image:main-latest-{KERNEL_ARCH}
#
#   2. `# arch:<csv>` per-line filter
#      Use this when each arch has a DIFFERENT tag (e.g. CI doesn't
#      publish a -amd64 build under the same naming convention).
#        foundationbot/foo:tag-arm64-only      # arch:arm64
#        foundationbot/foo:tag-amd64-only      # arch:amd64
#        foundationbot/foo:tag-shared          # arch:arm64,amd64
#
# Lines without `# arch:` apply to ALL arches.

# dma-ethercat installer image. The installer Job manifest at
# manifests/installers/dma-ethercat/base/job.yaml carries
# `foundationbot/dma-ethercat:PLACEHOLDER`; the real tag is operator-
# supplied via host-config.yaml's `images.dma-ethercat.image`.
# Bundling it here means phase 9 of bootstrap-robot.sh's installer
# Job pulls from local containerd (already imported by k0s) instead
# of DockerHub — works on robots without dockerhub-creds.
#
# Beta pin: SOF-1167. The two arch-filtered lines bundle the same
# SOF-1167 tag for both arm64 and amd64 robots. If DMA.ethercat's CI
# publishes arch-suffixed SOF-1167 builds (e.g. SOF-1167-aarch64 /
# SOF-1167-amd64) rather than a single multi-arch tag, adjust the
# tags below (and images.dma-ethercat in the host-config template).
foundationbot/dma-ethercat:SOF-1167    # arch:arm64
foundationbot/dma-ethercat:SOF-1167    # arch:amd64

# dma.streams beta pin (SOF-1167). The dma-recorder / rerun-streamer
# manifests still carry foundationbot/dma-streams:main-latest, so the
# manifest scan would otherwise bundle main-latest; this line ensures
# the SOF-1167 build the host-config override deploys is in the bundle.
foundationbot/dma-streams:SOF-1167
